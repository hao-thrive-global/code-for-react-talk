## How to run it

Make sure you have node installed on your machine
use `node -v` to check

1. use `npm` or `pnpm` to install dependencies for the app
   - `pnpm install` or `npm install`
2. run the app use `npm` or `pnpm`
   - `pnpm start` or `npm start`
